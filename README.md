# Ejemplos de Bash Scripting 🚀

Este repositorio contiene una colección de scripts básicos en Bash desarrollados como material de apoyo y guía para las clases de scripting.

## 📝 Contenido
Aquí encontrarás ejemplos prácticos que cubren:
* Uso de variables y lectura de datos (`read`).
* Argumentos por línea de comandos (parámetros posicionales).
* Estructuras condicionales (`if`, `elif`, `else`).
* Bucles e iteraciones (`for`, `while`).

## ⚙️ Cómo utilizarlos
1. Clona este repositorio o descarga el script que necesites.
2. Recuerda asignarle permisos de ejecución antes de correrlo:
   ```bash
   chmod +x nombre_del_script.sh

```

3. Ejecútalo en tu terminal:
```bash
./nombre_del_script.sh

```
